## 🔧 Common Issues and Fixes

**❌ GUI doesn't appear after executing**
> Confirm your executor injected successfully before running the script. Re-inject, rejoin the game, and execute again. On mobile, ensure overlay permissions are granted.

**❌ Auto roll runs but no auras register**
> The server may be experiencing high load. Try switching to a less populated server. Also confirm you're not hitting the game's built-in roll cooldown cap.

**❌ Auto collect potion misses spawns**
> This usually means the script version is behind the current map layout. Update to the latest release — potion node positions are patched regularly.

**❌ Auto craft produces no potions**
> Check that you have sufficient crafting ingredients in your inventory. The auto craft module requires raw materials to be present — it doesn't generate them.

**❌ Boss farm doesn't trigger encounters**
> The boss spawn may be on cooldown. The module waits for the cooldown to expire before triggering. If it's stuck, rejoin the server and re-execute.

**❌ Fishing bot casts but never reels**
> A game update may have changed the bite detection timing. Check the Issues tab for a reported fix or wait for the next patch update.

**❌ Anti AFK kicks still happening**
> Some servers have stricter idle detection. Increase the anti AFK pulse frequency in the GUI settings from the default interval to a shorter one.
